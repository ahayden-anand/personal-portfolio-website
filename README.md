#  Personal Portfolio Website

![Portfolio Preview](./public/assets/project1.png)


A modern and responsive personal portfolio website showcasing my skills, projects, and experience as a Computer Science Engineering student and aspiring software engineer.


##  About the Project

This portfolio website serves as a central place to present my technical background, projects, and professional journey. It reflects my focus on clean design, performance, accessibility, and real-world development practices.

The website is designed to be minimal, user-friendly, and responsive across all devices.

---

##  Features

-  Professional hero section with personal introduction  
-  Projects showcase with descriptions  
-  About section highlighting academic and technical background  
-  Fully responsive design (mobile-first)  
-  Contact section for easy communication  
-  Clean and modern UI  

---

## Tech Stack

- **React** 
- **Tailwind CSS**
